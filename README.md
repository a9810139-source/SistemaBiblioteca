# 📚 Sistema de Biblioteca (Java - POO)

## 📌 Descrição do Projeto

Este projeto é um sistema simples de biblioteca desenvolvido em Java, utilizando conceitos de Programação Orientada a Objetos (POO).

O sistema permite:
- Cadastro de livros
- Listagem de livros
- Consulta de livros por código
- Demonstração de herança com classe Aluno

---

## 🧠 Conceitos de POO utilizados

### ✔ Encapsulamento
Os atributos das classes são privados (`private`) e acessados por métodos getters.

### ✔ Herança
A classe `Aluno` herda atributos e métodos da classe `Pessoa`.

### ✔ Classes e Objetos
Foram criadas as seguintes classes:
- Livro
- Biblioteca
- Pessoa
- Aluno
- Main

### ✔ Métodos
Utilizados para:
- cadastrar livros
- listar livros
- consultar livros
- exibir dados do aluno

---

## 🏗 Estrutura do Sistema

- **Livro** → representa os livros da biblioteca
- **Biblioteca** → gerencia o cadastro e consulta de livros
- **Pessoa** → classe base
- **Aluno** → herda de Pessoa
- **Main** → classe principal que executa o sistema
